What have we done?

So far we have created a repository and added and cloned it to GitHub on the computer. We have also created an index.html and styles.css file to get started with coding.

The HTML file requires the doctype, html, head, and body tags. Additionally, it is important to define the character set with a meta tag.

In order for the correct styles to be applied to the page, the styles.css file needs to be specified in the index.html file with the link tag.

Committing!

  • git status - used to see the status of the files in the repo
  • git add <file> - used to commit specific files to the repo
  • git add -A -used to commit ALL files to the repo
  • git commit -m "message here" - used to add a commit message
